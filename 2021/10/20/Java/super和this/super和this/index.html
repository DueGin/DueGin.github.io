super和this

Author: DueGin Date: October 20, 2021  23:35:13 Category: java

super 和 this

this

this的作用

  • this调用本类属性
  • 调用本类方法
  • 利用this表示当前类对象

构造器

在构造器中，使用this来调用本类的函数，需要放在首位

super

  • 代表父类对象
  • 子父类存在着同名的成员时，在子类中默认时访问子类的成员，可以通过super关键字指定访问父类的成员
  • 创建子类对象时，默认会先调用父类无参的构造方法，可以通过super关键字指定调用父类的构造方法

super的用法

  • super.xx可以用来引用直接父类的实例变量。

  • super.xxx()可以用来调用直接父类方法。

  • super()可以用于调用直接父类构造函数

构造器

在构造器中，使用super来调用父类的函数，需要放在首位

注意事项

  • super 和 this不能同时出现在同一个构造函数中调用其他的构造函数，因为两个语句都要是第一个语句

this和super的区别

  • （一）代表的事物不同

    super代表的是父类空间的引用

    this代表的是所属函数的调用者对象

  • （二）使用前提不同

    super必须要有继承关系才能使用

    this不需要继承关系也能使用

  • （三）调用的构造函数不同

    super：调用父类的构造函数

    this：调用所属类的构造函数
